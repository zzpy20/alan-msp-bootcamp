# Tickets

Simulated helpdesk tickets for MSP practice. Each ticket follows a standard format to mimic real-world service desk workflows.

## Ticket Format

```
Ticket #: TKT-XXXX
Priority: Low / Medium / High / Critical
Category: Hardware / Software / Network / Account / Other
Status: Open / In Progress / Resolved / Closed

Description:
[User-reported issue]

Steps Taken:
[Troubleshooting steps]

Resolution:
[How the issue was resolved]

Time to Resolve:
```

## Categories
- `hardware/` — Printer, workstation, peripheral issues
- `software/` — App crashes, installs, licensing
- `network/` — Connectivity, VPN, DNS issues
- `account/` — Password resets, access requests, MFA
