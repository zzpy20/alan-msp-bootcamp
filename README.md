# Alan MSP Bootcamp

A structured learning repository for Managed Service Provider (MSP) skills — covering IT operations, helpdesk workflows, ticketing, networking, and cloud infrastructure.

## Repository Structure

```
alan-msp-bootcamp/
├── docs/          # Study notes, guides, and reference material
├── labs/          # Hands-on lab exercises and walkthroughs
├── portfolio/     # Completed projects and showcase work
└── tickets/       # Simulated helpdesk tickets for practice
```

## Goals

- Build practical MSP skills through structured labs
- Document knowledge and create a reference library
- Simulate real-world helpdesk and sysadmin scenarios
- Compile a portfolio of completed work
